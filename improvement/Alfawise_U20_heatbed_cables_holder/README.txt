# Summary

Heatbed cables holder for Alfawise U20 and clones like iformer Skykit :
- compatible with insulated heatbeds
- no contact with the heated parts (you can safely print it with PLA)
- compatible with stock corrugated cable sheath or any other replacement sleeve

Please put this holder between the red bed base and the adjustment knob.
